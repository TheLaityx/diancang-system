# 扫码点餐系统 - 部署文档

## 一、项目概述

本项目是一个基于微信原生小程序开发的扫码点餐系统，仿照瑞幸咖啡小程序设计风格，支持用户扫码点餐、商家订单管理等功能。

## 二、技术架构

- **前端**: 微信原生小程序 (WXML + WXSS + JS + JSON)
- **后端**: 微信云开发 (CloudBase)
- **数据库**: 微信云数据库 (MongoDB)
- **存储**: 微信云存储

## 三、部署步骤

### 3.1 开通云开发

1. 登录微信公众平台 (https://mp.weixin.qq.com/)
2. 进入小程序管理后台
3. 开通云开发环境（免费版即可）
4. 获取云开发环境ID

### 3.2 配置项目

1. 打开 `project.config.json` 文件
2. 修改 `appid` 为你的小程序AppID
3. 修改 `app.js` 中的云开发环境ID

```javascript
// app.js 第17行
wx.cloud.init({
  env: 'your-env-id', // 替换为你的云开发环境ID
  traceUser: true
});
```

### 3.3 创建数据库集合

在云开发控制台中创建以下集合：

1. **tables** - 桌台表
2. **categories** - 分类表
3. **dishes** - 菜品表
4. **orders** - 订单表
5. **refunds** - 退款表
6. **users** - 用户表
7. **merchants** - 商家表

## 四、功能说明

### 4.1 用户端功能

1. **扫码绑定桌号**
   - 扫描餐桌二维码自动绑定桌号
   - 桌号1-10对应10张餐桌

2. **浏览菜品**
   - 分类横向滑动
   - 搜索菜品
   - 菜品详情查看

3. **购物车**
   - 加入购物车
   - 数量调整
   - 删除菜品

4. **下单支付**
   - 确认订单
   - 模拟支付
   - 生成取餐号

5. **订单管理**
   - 查看订单列表
   - 订单详情
   - 申请退款

6. **我的页面**
   - 查看订单
   - 退款记录
   - 商家入口

### 4.2 商家端功能

1. **登录方式**
   - 账号密码登录（admin/123456）
   - 扫码登录（可选）

2. **订单管理**
   - 待接单列表
   - 接单/开始制作/完成
   - 退款审核

3. **菜品管理**
   - 添加/编辑/删除菜品
   - 上架/下架

4. **数据统计**
   - 今日订单数
   - 今日营收
   - 菜品销量排行

5. **系统设置**
   - 修改密码
   - 退出登录

## 七、注意事项

1. 本系统使用模拟支付，生产环境需对接真实微信支付
2. 需要配置小程序AppID才能正常运行
3. 云开发环境ID需要替换为实际环境ID
4. 图片资源需要自行准备或使用云存储上传

## 八、常见问题

**Q: 扫码后提示"无效的二维码"**
A: 检查二维码内容是否为 `?table=1` 格式

**Q: 商家登录失败**
A: 检查数据库中是否添加了商家账号数据

**Q: 菜品无法显示**
A: 检查数据库中是否添加了分类和菜品数据
